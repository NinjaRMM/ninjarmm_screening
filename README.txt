Building Step 1
---------------

The code for step 1 is contained in step1/src/main.cpp and it can be compiled with gcc as `g++ main.cpp -std=c++17 -o a` or using the CMake file. To use the CMake file, first create the a directory (named `build` for example) inside step1, and execute the following commands within this directory:
    1. `cmake ..`
    2. `cmake --build .`
